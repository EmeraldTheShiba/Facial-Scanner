# Automated Attendance Tracking System

A real-time facial recognition attendance system built with Python, InsightFace, OpenCV, and Supabase.

## Features

- Real-time face recognition using webcam
- Automated attendance logging
- Cloud database integration with Supabase PostgreSQL
- Unknown face detection and storage
- Basic GUI launcher
- Attendance log viewer

## Technologies Used

- Python
- OpenCV
- InsightFace
- Supabase
- PostgreSQL
- Tkinter
